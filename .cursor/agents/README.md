# Custom subagents (Cursor)

Cursor loads **custom subagents** from this directory as **individual `.md` files** (see [Subagents](https://cursor.com/docs/subagents)). The docs use the path `.cursor/agents/`; invoke with `/file-basename` (e.g. `/coordinator-agent`).

**Skills** (reusable workflows, folder + `SKILL.md`) live in [`.cursor/skills/`](../skills/README.md) — not here.
